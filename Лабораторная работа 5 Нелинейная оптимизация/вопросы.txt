

1.активный и пасивный поиск минимума функции
	Активный поиск минимума включает использование методов, которые активно изменяют параметры функции для нахождения её минимума. Примеры таких методов включают:

	Градиентный спуск: Метод, который использует градиент функции для нахождения направления наибольшего убывания и корректировки параметров в этом направлении.
	Метод Ньютона: Использует вторую производную (Гессиан) для более точного нахождения минимума.
	Методы сопряжённых градиентов: Оптимизируют функцию, используя информацию о предыдущих шагах для ускорения сходимости.

	Пассивный поиск минимума
	Пассивный поиск минимума включает методы, которые не требуют вычисления производных функции. Эти методы часто используются, когда функция сложна или производные трудно вычислить. Примеры включают:

	Метод случайного поиска: Исследует пространство параметров случайным образом и выбирает наилучший результат.
	Метод сеточного поиска: Разбивает пространство параметров на сетку и оценивает функцию в каждой точке сетки.
	Метод симплексного поиска (метод Нелдера-Мида): Использует геометрические фигуры (симплексы) для нахождения минимума функции.

как найти точное значение миниммум
Чтобы найти точное значение минимума функции, следуйте этим шагам:

Найдите производную функции. Это первый шаг, чтобы определить критические точки.
Приравняйте производную к нулю и решите уравнение. Это поможет найти критические точки.
Проверьте каждую критическую точку с помощью второй производной или теста на возрастание/убывание функции в окрестности точки. Если вторая производная положительна в этой точке, то это точка минимума.
Подставьте найденные критические точки в исходную функцию, чтобы найти точное значение минимума функции

методы первого нулевого второго порядка
Методы оптимизации можно классифицировать по порядку производных, которые они используют:

Методы нулевого порядка:
Эти методы используют только значения целевой функции, не требуя вычисления производных.
Примеры: метод Хука-Дживса, метод Нелдера-Мида12.
Методы первого порядка:
Эти методы используют значения функции и её первые производные (градиенты).
Примеры: градиентный спуск, метод сопряжённых градиентов34.
Методы второго порядка:
Эти методы используют значения функции, её первые и вторые производные (гессианы).
Примеры: метод Ньютона, квазиньютоновские методы13.
